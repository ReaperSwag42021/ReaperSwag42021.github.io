# reaperswag42021.github.io

Mijn persoonlijke portfolio-website, live via GitHub Pages op **[reaperswag42021.github.io](https://reaperswag42021.github.io/)**.

Een portfolio voor mij als Software Developer in opleiding bij Firda Leeuwarden — met mijn projecten, achtergrond en contactgegevens.

---

## 🧱 Gebouwd met

- **HTML5** — semantische, toegankelijke opmaak
- **CSS3** — custom properties, responsive layout, animaties
- **Vanilla JavaScript** — geen frameworks, geen build-stap
- **Google Fonts** — Syne & DM Sans
- **Web3Forms** — contactformulier zonder eigen server

Geen dependencies, geen bundler — gewoon statische bestanden die direct draaien.

---

## 🚀 Lokaal draaien

Omdat het pure statische bestanden zijn, kun je `index.html` gewoon in je browser openen. Voor een echte lokale server:

```bash
# Python 3
python -m http.server 8000
```

Open daarna `http://localhost:8000`.

## ✉️ Contactformulier

Het formulier op de contactpagina gebruikt [Web3Forms](https://web3forms.com). Vervang in `contact.html` de waarde van het `access_key` veld door je eigen sleutel om inzendingen te ontvangen.
