Ik ben Rik — een software developer uit Leeuwarden. Ik volg de opleiding Software Developer bij Firda en bouw daarnaast praktische digitale projecten waarin schone code en doordacht ontwerp samenkomen.

Mijn aanpak behandelt ontwerp en techniek als één discipline. Elke beslissing — van typografie tot componentarchitectuur — dient hetzelfde doel: ervaringen die vanzelfsprekend voelen.
